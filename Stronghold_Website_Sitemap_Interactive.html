Each page carries a priority (Critical High Medium). Future pages also carry a build verdict (Quick win Fast-follow Big bet Easy add Later Blocked) and three H/M/L scores with a stated reason. Pages marked New do not exist on the current Squarespace site.
Business Value (BV): revenue, growth, attribution, risk reduction. H is better. Supporter Value (SV): how much supporters asked for it / how it serves the relationship. H is better. Level of Effort (LOE): build + content cost. L is better (cheaper). Driven mostly by module reuse vs a new module build.
Voice rules (Brand Book): lead with "supporters"; "donor" is fine for segments and major-gift context · invitation never guilt · show one named story, never "we served 10,000" · dignity always · real field photography, never stock.
